# Flask IP Address Display App

A simple Flask web application that displays the user's IP address.

## Installation

1. Install the required dependencies:
```bash
pip install -r requirements.txt
```

## Running the Application

Run the application with:
```bash
python app.py
```

The application will be available at `http://localhost:5000`

## Usage

Visit `http://localhost:5000` in your browser to see your IP address displayed as "Your IP is: <ip>"
